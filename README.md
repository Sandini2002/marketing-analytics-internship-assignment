## â–¶ï¸ How to Run the Analysis

1. **Clone or download this repository.**
2. **Open the notebook** located in the `notebook/` folder:
* `customer_analytics_analysis.ipynb`


3. **Ensure the CSV files** (`customers.csv`, `products.csv`, `transactions.csv`) are located in the `data/` folder as shown in the structure above.
4. **Run the notebook cells** from top to bottom.

The notebook performs the following steps sequentially:

1. **Data Exploration & Cleaning:** Handling missing values (e.g., customer age, product ratings) and data typing.
2. **Feature Engineering:** Creating metrics like *Total Spend*, *Order Frequency*, and *Recency*.
3. **Customer Segmentation:** Grouping customers to identify high-value segments.
4. **Sales Analysis:** Visualizing trends and performance.

*All outputs and charts are generated directly within the notebook.*
